# Parhit Saris Welfare Trust  

**Parhit Saris Welfare Trust** is a non-profit organization dedicated to improving lives through education, healthcare, women empowerment, and sustainable community development.  

This repository serves as a digital hub for our projects and initiatives, providing open access to resources, updates, and collaboration opportunities.  

We are also leveraging modern technologies like **React** to build interactive web applications that enhance transparency, share impact stories, and connect with supporters more effectively.
